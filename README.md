# Chat Yard

A fullstack application Adonis & Vue

### Migrations

Run the following command to run startup migrations.

```js
adonis migration:run
```

### Assets

Transpile all assets and keep a watch over them with

```bash
npm run assets-watch
```

### Start the application

```bash
adonis serve --dev
```
